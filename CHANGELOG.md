# Changelog

## [1.0.0] - 2025-12-02
### Initial Release
- **Core Features:**
    - Client Management System
    - Project Management System
    - Invoicing & Estimates
    - Role-Based Access Control (RBAC)
    - Multi-Branch Support
    - Dark Mode Support

- **Advanced Chat System:**
    - Real-time messaging
    - Voice Message Recording & Playback
    - Audio File Upload & Playback (MP3, WAV, etc.)
    - Photo Sharing with Inline Previews
    - Document Sharing
    - Conversation Management (Delete, Clear History)
    - Project-linked Conversations

- **Technical:**
    - Built with Laravel 12 & Livewire
    - Tailwind CSS for styling
    - Alpine.js for interactivity
    - Optimized for performance
