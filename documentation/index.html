DigiCRM Documentation

Welcome to DigiCRM - The Ultimate Client Relationship Management System with Advanced Chat capabilities.

DigiCRM is a powerful, modern, and secure web application designed to help businesses manage clients, projects, invoices, and team communication efficiently. Built with Laravel 12 and Livewire, it offers a seamless single-page application (SPA) experience.

Server Requirements

Before installing DigiCRM, ensure your server meets the following requirements:

  • PHP >= 8.2
  • MySQL >= 5.7 or MariaDB >= 10.3
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

Installation Guide

1. Upload Files

Unzip the downloaded package and upload all files to your server's public directory (e.g., public_html).

2. Database Setup

Create a new MySQL database via your hosting control panel (cPanel, Plesk, etc.). Note down the database name, username, and password.

3. Environment Configuration

Rename the .env.example file to .env. Open it and update the database credentials:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_database_name
DB_USERNAME=your_database_user
DB_PASSWORD=your_database_password

4. Install Dependencies & Migrations

Run the following commands in your terminal (SSH):

composer install
npm install
npm run build
php artisan key:generate
php artisan migrate --seed
php artisan storage:link

The --seed command will create a default Super Admin account.

Configuration

Mail Setup

To enable email notifications, configure your SMTP settings in the .env file:

MAIL_MAILER=smtp
MAIL_HOST=smtp.mailtrap.io
MAIL_PORT=2525
MAIL_USERNAME=your_username
MAIL_PASSWORD=your_password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS="hello@example.com"
MAIL_FROM_NAME="${APP_NAME}"
